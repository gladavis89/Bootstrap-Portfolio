Psychic Game

The computer generates a random letter. You have 9 guesses to match the correct letter. As you guess, the screen will show the letters you have input and the guess counter will count down. If you win or lose, an alert is generated and the wins or losses counter will increase by 1.

Try it out!

Crystal Collectors

This game is much like the psychic game but uses jQuery and has a much nicer user interface. The computer generates multiple random numbers. The object is to add up the numbers generated on the collectors to your given score. Winning or losing will increase your score and reset the generated numbers.

Try it out!
